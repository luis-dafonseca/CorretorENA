
# Download do CorretorENA

## Baixar o programa

Selecione a opção adequada para seu sistema:

- Versão _standalone_ para [Windows](https://github.com/luis-dafonseca/CorretorENA/releases/download/v0.1.0-alpha/windows-latest-v0.1.0-alpha.zip)
- Versão _standalone_ para [Linux  ](https://github.com/luis-dafonseca/CorretorENA/releases/download/v0.1.0-alpha/ubuntu-latest-v0.1.0-alpha.zip)
- Versão _standalone_ para [macOS  ](https://github.com/luis-dafonseca/CorretorENA/releases/download/v0.1.0-alpha/macos-latest-v0.1.0-alpha.zip)

Se preferir baixar o código-fonte e executá-lo diretamente,
siga estes [passos](executar_codigo.md).

O programa é _standalone_, o que significa que não necessita
de instalação: basta descompactar o arquivo e executá-lo
diretamente após o download.

Ao executar um programa _standalone_, o sistema operacional pode
exibir uma mensagem solicitando confirmação antes de permitir a
sua execução. Esse comportamento é comum em Windows, macOS e
algumas distribuições Linux, pois esses sistemas têm mecanismos
de segurança que verificam a origem do executável.
Caso essa mensagem apareça, basta confirmar que o arquivo é
confiável para prosseguir com a abertura.
